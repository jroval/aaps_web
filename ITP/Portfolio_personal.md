# Portfolio 

## Proyecto – Proyecto AA Property Services

AA Property Services es una plataforma diseñada para gestionar propiedades, incidencias y servicios asociados a villas, simulando el funcionamiento real de una empresa de mantenimiento.

## Explicación

El proyecto está dividido en varias partes:

- Web corporativa (HTML + CSS)
- Base de datos diseñada desde cero (MySQL)
- Aplicación Java conectada mediante JDBC
- Sistema de gestión con funcionalidades CRUD

La aplicación permite gestionar propietarios, villas e incidencias, simulando un entorno real de trabajo.
