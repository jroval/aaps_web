 Presentación del proyecto – AA Property Services

## ¿Qué es el proyecto?

AA Property Services es una plataforma de gestión de propiedades orientada a villas de alto nivel, que centraliza el mantenimiento, las incidencias y los servicios asociados a cada propiedad.

El proyecto simula el funcionamiento real de una empresa que gestiona múltiples viviendas, ofreciendo tanto servicios básicos de mantenimiento como servicios premium.

## ¿Qué problema resuelve?

En el sector inmobiliario, especialmente en propiedades de lujo, existe una necesidad constante de:

- Supervisión del estado de la vivienda
- Gestión de incidencias técnicas
- Coordinación de servicios (limpieza, jardinería, reparaciones, etc.)
- Comunicación clara entre propietario y empresa

Este proyecto resuelve ese problema centralizando toda la información y operativa en una única aplicación.

## ¿Para quién está pensado?

- Propietarios de villas que quieren controlar el estado de su vivienda
- Empresas de mantenimiento que gestionan múltiples propiedades
- Equipos internos que necesitan organizar tareas y servicios

## ¿Qué tecnologías utiliza?

- Java (aplicación principal)
- JDBC (conexión con base de datos)
- MySQL (gestión de datos)
- HTML y CSS (portal web corporativo)
- Git y GitHub (control de versiones y documentación)

## ¿Qué funcionalidades incluye?

- Gestión de propietarios
- Gestión de villas
- Sistema de incidencias
- Sistema de login por consola
- Operaciones CRUD completas sobre base de datos
- Arquitectura organizada en capas (controller, service, DAO)

## ¿Qué he aprendido con este proyecto?

- Diseñar una base de datos realista basada en un negocio
- Conectar una aplicación Java con una base de datos mediante JDBC
- Implementar operaciones CRUD reales
- Estructurar un proyecto siguiendo principios de programación orientada a objetos
- Separar responsabilidades en distintas capas (arquitectura limpia)
- Trabajar con GitHub de forma profesional
- Pensar como desarrollador en un entorno real, no solo académico
